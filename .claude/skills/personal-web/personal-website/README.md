# 冷冰的个人主页

一个简约、清爽且具有科技感的个人主页，包含数字分身聊天功能。

## 功能特点

- **个人信息展示**：展示个人简介、职业、兴趣、擅长领域等
- **作品集展示**：展示代表性作品和项目
- **数字分身聊天**：AI驱动的聊天机器人，可以回答关于冷冰的问题
- **完全响应式设计**：适配手机、平板和桌面设备
- **现代化界面**：深蓝色和白色配色方案，简约科技风格

## 文件结构

```
personal-website/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 聊天功能JavaScript
├── avatar.svg          # 头像图像
└── README.md           # 说明文档
```

## 如何预览

### 方法1：直接打开（推荐）
直接在浏览器中打开 `index.html` 文件。

### 方法2：使用本地服务器
如果你有Python，可以运行：
```bash
python -m http.server 8000
```
然后在浏览器中访问 `http://localhost:8000`

## 设计特色

1. **配色方案**：深蓝色(#1e3a8a)和白色为主色调，体现科技感
2. **响应式布局**：
   - 桌面端：两栏布局（个人信息 + 聊天区）
   - 平板端：自适应调整
   - 手机端：单栏垂直布局
3. **交互效果**：
   - 悬停动画
   - 平滑过渡
   - 打字机效果的聊天回复
4. **聊天功能**：
   - 预定义问题建议按钮
   - 智能回答匹配
   - 打字动画效果

## 聊天功能知识库

数字分身可以回答以下类型的问题：
- 个人基本信息（职业、兴趣、特点）
- 近期工作和项目
- 作品集相关内容
- 联系方式
- 网站技术相关
