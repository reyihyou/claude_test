# 冷冰的个人主页

一个简约、清爽且具有科技感的个人主页，包含数字分身聊天功能。

## 功能特点

- **个人信息展示**：展示个人简介、职业、兴趣、擅长领域等
- **作品集展示**：展示代表性作品和项目
- **数字分身聊天**：AI驱动的聊天机器人，可以回答关于冷冰的问题
- **完全响应式设计**：适配手机、平板和桌面设备
- **现代化界面**：深蓝色和白色配色方案，简约科技风格

## 文件结构

```
personal-website/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 聊天功能JavaScript（支持AI+本地双模式）
├── avatar.svg          # 头像图像
├── README.md           # 说明文档
└── backend/            # DeepSeek API代理服务
    ├── server.js       # Express后端服务器
    ├── package.json    # Node.js依赖配置
    ├── .env.example    # 环境变量示例
    └── README.md       # 后端使用说明
```

## 如何预览

### 方法1：直接打开（推荐）
直接在浏览器中打开 `index.html` 文件。

### 方法2：使用本地服务器
如果你有Python，可以运行：
```bash
python -m http.server 8000
```
然后在浏览器中访问 `http://localhost:8000`

## AI聊天功能集成

现在支持真实的DeepSeek大模型聊天功能！

### 后端服务配置

项目包含一个Node.js后端服务，位于 `backend/` 目录，用于安全调用DeepSeek API：

```
backend/
├── server.js           # Express后端服务器
├── package.json        # Node.js依赖配置
├── .env.example        # 环境变量示例
└── README.md           # 后端使用说明
```

### 启动完整服务

1. **启动后端服务**：
   ```bash
   cd backend
   npm install
   npm run dev
   ```
   服务将在 `http://localhost:3000` 启动

2. **启动前端服务**：
   ```bash
   python -m http.server 8000
   ```

3. **访问页面**：
   - 打开 `http://localhost:8000`
   - 聊天功能将自动连接后端AI服务

### 功能特点

- **真实AI对话**：基于DeepSeek大模型的智能回复
