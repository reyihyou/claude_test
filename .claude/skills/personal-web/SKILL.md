# 个人主页
我想做一个个人主页，上面有一个数字分身聊天区。

## 【关于我】
- 名字：冷冰
- 一句话介绍：一个正在学习用 AI 做产品的内容策划
- 我现在主要在做：整理自己的作品和写作方向
- 我的兴趣：AI 应用、写作、旅行
- 我一个比较有记忆点的特点：喜欢把复杂问题讲成人话

## 【这个主页给谁看】
- 主要访客：朋友、潜在合作伙伴、面试官

## 【页面里至少要有】
- 头像
- 名字和一句话介绍
- 一个简单的个人信息展示区
- 一个可以提问的数字分身聊天区

## 【数字分身需要知道】
- 我的职业或身份：内容策划
- 我最近在做的事：搭自己的个人主页，整理作品集
- 我擅长或关心的方向：内容表达、AI 应用、知识整理
- 别人最可能问我的 3 个问题：你现在在做什么？你有哪些作品？怎么联系你？

## 【风格要求】
- 我希望整体感觉是：简约、清爽、稍微有一点科技感
- 我更偏好的颜色：深蓝色和白色
- 是否要适配手机：是
- 其他要求：不要太花哨，信息层次要清楚
